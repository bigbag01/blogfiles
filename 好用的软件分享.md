## typora

**简介** ：markdown编辑器。typora是windows下我目前用过的感觉最棒的markdown编辑器，现在博客基本都是在这里编辑。

**优点**：

1. 所见即所得，行内预览，书写感觉非常流畅。
2. 插入图片、表格、数学公式等都非常方便，有菜单栏可选，熟悉了以后也可以直接用快捷键。
3. 直接附带转为pdf和html的功能，如果配合pandoc一起用，还可以直接将markdown转化为docx、epub等格式
4. 配套theme可定制，程序员们可以用css写一套喜欢的主题
5. 因为是用js构建的，界面其实是内嵌了chromium浏览器，所以typora是跨平台的。
6. 免费

**缺点**： 

1. 单机，无同步功能。然而我把笔记都放到博客上了，直接打开网址就可以看了
2. 页面为js渲染，打开比较慢




## wox

**简介**： 开源快速启动工具，类似于mac上的alfred。默认唤起方式是 alt + space，可以更改

**优点**： 

1. 用wox非常方便直接由键盘操作启动软件，任务栏和桌面不放快捷方式也完全没问题。
2. 支持拼音匹配
3. 可以直接在wox里写简单的dos命令，命令的打开方式默认为cmd，也可以在设置里改成powershell
4. 配合**everything** 使用找东西超级快
5. 可以直接当简单的计算器来用，可以查16进制值对应的颜色。
6. 有一些配合的插件，比如词典。我习惯用有道，直接输入yd powerful 就可以查到powerful的翻译，中英互译皆可。
7. 免费！开源！可以自己定制！

**缺点**

1. 还不是很完善，偶尔会出bug，不过目前为止遇到的问题都是直接重启它就好




##Everything

**简介**： 本地搜索神器。有了它谁还用资源管理器自带的搜索╭(╯^╰)╮

**优点**：

1. 超级快！超级快！超级快
2. 体积小巧，占用资源少
3. 支持通配符和正则表达式
4. 免费

**缺点**： 

1. 每次重启电脑之后打开everything需要花1分钟左右建立索引。




## Sublime Text3

**简介**： 文本编辑器

**优点**：

1.  打开文件速度快。有时候不需要运行一个工程，只是看看它的结构和代码，直接用sublime非常快。而打开一个IDE的时间，都可以出去走一圈了。
2.  插件多，需要什么功能基本都可以找到配合的插件，可以配置成为js，python等脚本语言的IDE
3.  免费

**缺点**：

1. 虽然免费，但是经常会提醒你可以选择购买许可证，有点烦
2. 作为脚本语言IDE还是没有专用的IDE比如pyCharm,webStorm等功能丰富。




## 奇妙清单（wunderlist）

**简介**： 备忘录。

**优点**：

1. 可多平台同步
2. 待办事项可以设置提醒，可以分类，可以为任务设置子任务
3. 界面好看
4. 高级账户收费，普通账户是免费的，就我目前所用来看，普通账户够用了。

**缺点** ：

1. 它会让你发现……啊，任务好多做不完::>_<::
2. uwp版好像有时候同步会出错？




## 美图秀秀批处理

**简介**： 图片批处理

**优点**：

1. 批处理图片非常方便
   * 可以一次性对导入的所有图片做亮度对比度尺寸等的调整
   * 保存图片时按顺序可以用数字重新命名，保存格式也可以一起更改。
2. 在写图像处理的程序的时候，经常需要对输入的图片进行预处理，写脚本是很方便，然而用这个更快一点。
3. 免费！

**缺点**： 

1. 独立版好像没有官方下载渠道，要和美图秀秀一起下载。




## tortoise git

**简介**： git 客户端

**优点**： 

1. 操作简单，不用再去记git命令了
2. diff查看版本变化非常方便醒目
3. 开源，免费

**缺点**：

1. 配置安装的过程要按着教程做，否则就一脸懵逼……




## chrome

**简介**： google出的浏览器

**优点**： 

1. chrome有很多扩展和应用，下一篇博客会介绍我用过的觉得很棒的浏览器插件
2. 速度快
3. 前端调试方便（没怎么用过firefox，不知道和firebug比具体如何）
4. 多平台同步方便，一个Google账户，手机平板电脑上的chrome浏览器的书签cookies啥的都同步了。

**缺点**：

1. 占用内存厉害
2. google被墙，导致一些帮助页面打不开。下载的时候，最好要找离线安装包。




## TeamViewer

**简介**： 远程桌面控制软件

**优点**：

1. 操作简单便捷。只要在两台机器上同时运行teamviewer，然后输入对方的ID就可以建立连接。
2. 安全性好
3. 支持多平台。可以用手机、平板控制电脑，也可以反过来。在开会或者上课带电脑比较重，又需要查看电脑里的文件的时候，用这个非常方便
4. 免费。收费版增加一部分功能，然而也都并不常用。

**缺点**： 

1. 免费版好像每次关的时候会跳广告……直接关掉就好。
